Welcome to BloofMC's documentation!
===================================

**BloofMC** is a Minecraft Server Network for Adults
that is geared towards longterm play and reliability.
The network is made up of five servers that are available at all times.
Servers available 24/7: Survival, Vanilla, Creative, SkyBlock, and OneBlock.

This documentation will be your guide to success.
Here you can learn everything about the network and each of its servers!

.. note::

   This project is under active development.
